# Soul Heart Here

Soul Heart Here is an emotion-first support website created for people who feel lonely or emotionally stressed and need someone to listen.

## Purpose
To provide a calm, safe, and human-centered space where users can reach out and feel understood.

## Features
- Single-page smooth scrolling design
- Emotional and calming visuals
- "I'm Ready to Talk" action button
- Mandatory email and phone validation
- Trusted / rating section
- Contact section with map location
- Fully responsive for mobile and desktop

## Tech Stack
- HTML5
- CSS3
- JavaScript
